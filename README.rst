uData Administration

Compatibility
=============

udata-admin requires Python 2.7+ and uData.


Installation
============

You can install udata-admin with pip:

.. code-block:: console

    $ pip install udata-admin

or with easy_install:

.. code-block:: console

    $ easy_install udata-admin
