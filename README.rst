Compatibility
=============

udata-harvest requires Python 2.7+ and uData.


Installation
============

You can install udata-harvest with pip:

.. code-block:: console

    $ pip install udata-harvest

or with easy_install:

.. code-block:: console

    $ easy_install udata-harvest
